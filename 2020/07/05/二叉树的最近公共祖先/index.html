二叉树的最近公共祖先

最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

解题思路：

  • 使用先序遍历的思路
  • 遍历时若节点为 p或q,返回当前节点，若节点为null ,返回null(叶子节点时，没有左右节点会返回null)，若不是，递归左右节点
  • 判断递归结果，若左节点或者右节点不为null（表示匹配到节点），返回匹配的左右节点；若节点为左右节点都为null，返回null

复杂度计算：
时间复杂度O(N)：p,q都在叶子节点时，要遍历整棵树,每个结点最多遍历一次
空间复杂度O(N):二叉树最坏情况下为一条链，此时高度为 N，因此空间复杂度为 O(N)
